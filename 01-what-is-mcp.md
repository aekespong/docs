# What is MCP?

Large language models like Gemma are remarkable — they can reason, write, translate, and explain. But they have a fundamental limitation: **they only know what they were trained on.**

Ask an AI what time it is right now, and it cannot tell you. Ask it for today's stock price, the latest news, or data from your company's systems — and it is stuck. The AI's knowledge has a cutoff date, and it has no way to reach out to the world around it.

## Enter MCP

**MCP stands for Model Context Protocol.** It is an open standard that gives AI models a structured way to access external tools and data sources in real time.

Think of it like a universal integration solution. Instead of every AI application building its own custom wiring to connect with every external service, MCP provides a single, agreed-upon interface. Any Generative LLM that supports agents speaks MCP can connect to any MCP server — and instantly gain new capabilities.

## What MCP Makes Possible

With MCP, an AI assistant can:

- Look up live data from an external API
- Search a database or document store
- Trigger actions in other systems

The AI does not need to be retrained to gain these abilities. It simply needs to be told what tools are available — and MCP defines exactly how that conversation happens.

## In This App

This chat connects to one or more MCP servers. You can see which servers and tools are active in the **MCP Servers** panel on the right. The AI can call those tools in real time as part of answering your questions.
